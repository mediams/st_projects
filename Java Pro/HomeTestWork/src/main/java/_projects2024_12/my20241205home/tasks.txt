Я начал изучение в Джаве ввод вывод запись файл чтение с файла а также блоки try catch

Дай мне пять задание на эту тему на понимание этой темы от простого к сложному
-

Задание 1: Запись строки в файл
Описание:
Напишите программу, которая создает текстовый файл output.txt и записывает в него строку "Привет, мир!". Убедитесь, что файл создается в корневой папке вашего проекта.

Подсказка: Используйте классы FileWriter и блок try-catch.

Задание 2: Чтение строки из файла
Описание:
Создайте текстовый файл input.txt, добавьте туда несколько строк текста (например, "Привет, Java!\nКак дела?"). Напишите программу, которая считывает содержимое файла и выводит его в консоль.

Подсказка: Используйте FileReader и BufferedReader.

Задание 3: Копирование содержимого из одного файла в другой
Описание:
Напишите программу, которая считывает содержимое файла source.txt и записывает его в новый файл destination.txt. Если файл source.txt не существует, программа должна вывести сообщение об ошибке и завершить работу.

Подсказка: Используйте блок try-catch для обработки исключений и работу с файлами.

Задание 4: Подсчет количества строк в файле
Описание:
Напишите программу, которая считывает текстовый файл data.txt и подсчитывает количество строк в нем. Если файл пустой или отсутствует, программа должна вывести соответствующее сообщение.

Подсказка: Используйте цикл для построчного чтения файла.

Задание 5: Обработка чисел в файле
Описание:
Создайте текстовый файл numbers.txt, в котором на каждой строке будет одно число (например, 1\n2\n3\n4\n5). Напишите программу, которая:

Считает сумму всех чисел в файле.
Если в файле встречается нечисловое значение, программа должна вывести сообщение об ошибке, указав строку, где произошла ошибка, но продолжить обработку оставшихся строк.
Подсказка: Используйте try-catch внутри цикла для обработки ошибок формата чисел.
